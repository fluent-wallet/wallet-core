# WalletCore 项目概述

WalletCore 是一个模块化的区块链钱包核心库，提供了完整的钱包功能实现，支持多链操作。

## 核心特性

- 多链支持：BTC、ETH 等主流公链
- 安全的密钥管理
- 可扩展的模块化设计
- 统一的数据存储接口
- 完整的交易生命周期管理

## 架构设计

### 核心层 (Core)

核心层提供基础功能实现：

- 密钥管理与签名
- 钱包账户管理
- 网络通信抽象

### 链层 (Chains)

为不同区块链提供具体实现：

- 地址生成
- 交易构建
- 链特定功能

### 数据层 (Data)

处理数据存储和管理：

- 统一的数据模型
- 可插拔的存储后端
- 缓存管理
